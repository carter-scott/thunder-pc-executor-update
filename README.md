## About Thunder Executor

Thunder Executor is a free Roblox script executor for Windows that does not require a key. It is designed to stay lightweight, keeping the install size under 30 MB while still offering a clean workflow for Lua script injection. From casual gameplay tweaks to Lua testing, the app provides a straightforward execution environment and includes a built-in script hub with more than 500 community-made scripts.

To reduce maintenance, the program includes an auto-update system that helps keep compatibility aligned with the newest Roblox client releases, so manual patching is not needed. The interface also supports multiple languages and lets you inject with a single click, making everyday use simple. A persistent queue powered by SQLite preserves your saved script selections between launches, which is useful if you return to the same setup often.

**Does it work with the latest Roblox update?**  
Yes. The auto-update engine is intended to keep the executor aligned with current Roblox releases. If a new update disrupts compatibility, a patch should be fetched automatically the next time you start the app.

**How does Thunder Executor compare to other free executors?**  
It stands out by avoiding key systems, keeping a persistent script queue, and including a built-in debugger. Those capabilities are commonly associated with paid tools.

**Can I get banned for using this executor?**  
Any third-party script executor can violate Roblox's terms of service and may lead to account suspension. Thunder Executor does not provide any guarantee against bans.

**Where are my scripts stored locally?**  
Your scripts live in the `scripts/` folder, while the saved queue is kept in a SQLite database under the `hub/` directory.
